# figures/

Plots exported by the notebooks (`.png`). Each notebook saves its figures here via
relative paths (`../../figures/...`). Figures are regenerated on every run.

Generated figures include:

- `logistic_Perlidae.png`, `logistic_Helicopsychidae.png` — fitted logistic curves.
- `tree_Perlidae.png`, `tree_Helicopsychidae.png` — classification trees.
- `negative_binomial_BMWP.png` — fitted negative binomial curve (BMWP vs hardness).
- `bmwp_spearman.png` — real vs fuzzy-predicted BMWP per station with Spearman's rₛ.
